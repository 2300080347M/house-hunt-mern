# 🏠 House Hunt – MERN Stack House Rental Platform

A full-stack MERN web application that allows users to explore, create, and manage rental property listings with secure authentication and real-time property browsing.

---

# 🚀 Features

- 🔐 User Authentication (Sign In / Sign Up)
- 🏡 Create & Manage Property Listings
- 📸 Multiple Image Uploads
- 🔎 Advanced Property Search & Filters
- 👤 User Profile Management
- 📩 Buyer–Owner Contact through Email
- 🗄️ MongoDB Database Integration
- ⚡ Responsive Modern UI
